# 快捷键测试指南

## 永久快捷键测试
- V - 选择工具
- R - 矩形工具
- E - 椭圆工具
- B - 贝塞尔工具
- Y - 折线工具
- Shift+P - 多边形工具
- Shift+B - 画笔工具
- F - 填充工具
- Shift+G - 渐变填充工具
- P - 钢笔工具
- Shift+E - 橡皮擦工具
- L - 直线工具
- Ctrl+Shift+P - 路径编辑工具
- N - 节点编辑工具（仅在有选中对象时有效）

## 临时快捷键测试（按住切换，松开恢复）
- X - 临时切换到画笔工具
- C - 临时切换到橡皮擦工具
- D - 临时切换到填充工具
- Space - 临时切换到选择工具
- Q - 临时切换到钢笔工具
- W - 临时切换到矩形工具
- A - 临时切换到椭圆工具
- S - 临时切换到直线工具

## 上下文相关快捷键测试
- N键只在有选中对象时才会切换到节点编辑工具

## 测试步骤
1. 启动应用程序
2. 尝试各种永久快捷键，验证工具切换
3. 按住临时快捷键（如X），验证临时切换到画笔工具
4. 松开临时快捷键，验证是否恢复到之前的工具
5. 创建一些图形，选中它们，然后测试N键的上下文功能
6. 在没有选中对象时测试N键，应该不会切换到节点编辑工具

## 预期行为
- 临时工具切换应该在1秒后自动恢复原工具
- 临时快捷键应该优先于永久快捷键
- 上下文相关的快捷键只在满足条件时生效
- 不应该有快捷键冲突警告
